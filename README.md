## Пользователи и группы Авторизация и аутентификация

Среда выполнения ДЗ: VM Centos 8 в VMware vSphere 7

### Файлы:
```
- newvm.yml:    ansible playbook развертывания VM,
                настройки подключения по ssh,
				установки docker,
				предоставления работы с docker группе admin

- login.sh:     скрипт /usr/local/bin/login.sh
                Скрипт работает по принципу: 
                если сегодня среда или четверг, то нужно проверить,
				входит ли пользователь в группу admin,
				если не входит — то подключение запрещено.
				При любых других вариантах подключение разрешено

- ssh.sh:       скрипт настройки подключения по ssh,
                создания группы, пользователей

10-docker.rules:файл /etc/polkit-1/actions/10-docker.rules
                правило, позволяющее пользователям группы admin,
				рестартовать docker.service

- ansible.jpg:  скриншот работы ansible
- ssh.jpg:      скриншот, показывающий возможность подключения по ssh
                в среду и четверг только группе admin
                        
- docker.jpg    скриншот, показывающий возможность работать с docker
                и рестартовать docker.service группе admin
```
